Glass Mask Icon Pack

📌 About

I used the Icon Packer app to create this app.
 
Why this icon pack is different?

Most icon packs provide custom icons for selected apps, and users often need to request missing icons from the developer.  
This pack solves that problem: it does not provide individual icons for any app. Instead, it applies a glass masking effect to all icons on the device.  
That means every app icon gets a consistent glass mask style automatically—no requests, no waiting, no missing icons.
